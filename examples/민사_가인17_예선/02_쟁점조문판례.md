## 쟁점별 결론 구조 (원고측)

1. **가등기 = 담보가등기**: 제1차 확인서 "차용" 명시(6항) + 배당절차 쌍방 행태(9항) + 91다36932 실질 기준 → 가담법 §13 저당권 의제 (설정일 2015.12.5.)
2. **법정지상권 전면 부정** — 3중 안전망:
   - 담보가등기(=저당권) 설정 시(2015.12.5.): 나대지 → 2003다26051
   - 순위보전 가등기라도: 선행 가압류 시(2016.9.11.) 기준(2024다236327) → 나대지
   - 압류 시(2023.9.10.) 기준이라도: 기초공사만 → 독립건물 X(2000다51872), 건축 중 법리 요건 미충족(2003다29043)
3. **피고별**: 정대박 = 미등기 소유자 → 철거 + 부지 점유 부당이득(2025.2.26.~); 송운수 = 사실상 처분권자·직접점유자(86다카1751) → 철거 + 인도 + 부당이득(2025.9.10.~, 정대박과 부진정연대)
4. **항변 반박**: 선의점유(민197② 악의 의제 + 등기부 공시), 권리남용(정당한 소유권 행사), 96다31895(합의해제 사안 — 구별)
