# khulaw-record — 기록형/가인 대회 풀이 toolkit

가인법정변론대회·변호사시험 기록형(사건기록) 풀이 및 참고서면 작성을 위한 **자기완결형** 룰·프롬프트·스킬 모음. 대회장에서 이 repo를 pull해 그대로 사용하는 것을 전제로 한다.

> **비공개(Private) 유지 필수.** 풀이 전략·toolkit 노출 방지. 예시 사건기록은 가상 경진대회 기록(마스킹)이나, 외부 공개·공유하지 않는다.

## 구성

| 폴더 | 내용 |
|---|---|
| `rules/` | 기록형 풀이 핵심 룰(R1~R10) + 자료보강 기준 |
| `prompts/` | 기록형 풀이 프롬프트, 참고서면 작성 룰·프롬프트 |
| `skills/기록형풀이/` | 풀이 스킬 정의(SKILL.md) |
| `pilot/` | 파일럿 적용 사례(2026형제8765) |
| `examples/형사_2026형제8765/` | 형사 사건기록 예시(고소장·진술조서·의무기록 등, 마스킹) |
| `examples/민사_가인17_예선/` | 민사 예선 자료 예시(사실관계·쟁점·참고서면) |
| `AGENTS.md` | 도구가 세션 시작 시 읽는 근거룰(lean) |

## 사용 순서 (대회/연습 공통)

1. `AGENTS.md` → `rules/기록형풀이_룰.md` 로드.
2. 사건기록을 `examples/` 형식처럼 투입(또는 대회 제공 기록 사용).
3. `prompts/기록형풀이_프롬프트.md` 적용 → 쟁점·법리·포섭·서면 초안.
4. 조문·판례 사건번호는 korean-law-mcp(`.mcp.example.json` 참고) 또는 `rules/기록형풀이_룰.md`의 사이트 매핑으로 검증.

## 원칙 요약

- **기록 한정**: 사건기록에 있는 사실만 사용, 창작 금지.
- **인용 의무**: 사실=기록 발췌+위치, 법리=조문 원문/판례 사건번호.
- **실재 판례만**: 미확인 사건번호는 `[검증필요]`.
- **한국어·한자→한글**, 추론 비노출, 법원 제출 문체.

상세는 `rules/기록형풀이_룰.md`(R1~R10) 참조.

## 출처

이 toolkit은 개인 학습 워크스페이스(khulaw)에서 분리한 스냅샷이다. 학습용 본체는 별도 비공개 repo에서 유지되며, 대회 전 최신본으로 갱신해 사용한다.
